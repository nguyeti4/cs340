QuizQuestions

This page is used to manage the QuizQuestions table. Normally, the information of this table should come from the customer facing website. Each time a user took a quiz, all the questions of that quiz will be inserted into this table. For using this page, admin can check the detail of a particular quiz if something went wrong. Also, for the purpose of statistic, the admin staff can calculate the rate of correctness for each question.
